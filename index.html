Vue

Vue.js基本使用
常用的几个指令
template: 可以手动定义模板 <不推荐使用>
<标签 v-text=”数据名”></标签> 或者 <标签></标签> 文字
<标签 v-html=”数据名”></标签> html标签
<标签 v-bind=”数据名”></标签> 添加属性

v-on:事件类型=函数名 绑定事件
注意：用mothods方法 里面写函数处理方式 不带前缀on

指令缩写
v-bind:属性 -> :属性
v-on:事件类型 -> @事件类型

Vue中的this指向？
this指向Vue对象

事件修饰符
@事件.stop=”函数名” 阻止冒泡事件
@事件.pervent = “函数名” 阻止a连接的默认行为
@事件.once = “函数名” 该事件只能触发一次

v-model表单数据绑定(双向数据绑定)
v-model 类似于表单

methods与computed的区别
Methods中的方法每执行一次或者调用执行函数体代码里面的数据都会一一的执行下来
Computed中的计算属性 如果原始的数据没有发生变化 只会第一次完整的执行一次其余的都是拿计算属性的缓存输出

监听器
Watch:{
需要监听的函数:function(){
}
}

数据的遍历
<标签 v-for=”item in list”></标签>
v-if 与 v-show 类似于display false为隐藏 true为显示
建议使用v-show
